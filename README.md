# simplefeed-frontend

Frontend for `simplefeed`.

This project was bootstrapped via CRA, but was migrated to vite later.

This project does not claim to be finished, complete or thoroughly tested.

## Getting started

0. Run `npm i`
1. Start via `npm run start` which will run `vite` on port 5173.

# Deploy

This project can be deployed via vercel or similar hosters.
